# Eventblend Frontend

## Setup

### Install dependencies

```
pnpm i
```

### Add env variables

```
cp .env.example .env
```

### Start dev server

```
pnpm run dev
```

Also setup eventblend server locally
https://github.com/harshmangalam/event-blend-server

### Open

http://localhost:5173/

## Tech stack

- Typescript
- Qwik
- Qwikcity
- Qwik UI
- Tailwindcss
- Valibot
- Wretch
- Modular forms
